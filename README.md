# AIronVision

Análisis de técnica de entrenamiento de hipertrofia con computer vision e IA.

## Estructura

- `backend/` — API FastAPI (monolito modular)
- `mobile/` — App Flutter (iOS + Android)
- `ml/` — Notebooks de experimentación con modelos de CV
- `infra/` — Docker, scripts de deploy
- `docs/` — Documentación técnica y de producto

## Stack

- **Backend:** Python 3.12 + FastAPI + uv
- **Mobile:** Flutter
- **Auth + DB + Storage:** Supabase
- **Computer Vision:** MediaPipe Pose + OpenCV
- **Deploy:** VPS Hostinger + Coolify
