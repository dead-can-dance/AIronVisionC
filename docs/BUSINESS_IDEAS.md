## Tiers de monetización (idea inicial — abr 2026)

- **Free:** N análisis batch al mes, ejercicios básicos.
- **Pro:** análisis batch ilimitado, todos los ejercicios, histórico, comparativas.
- **Premium / Coach Mode:** análisis en tiempo real con cámara, feedback durante el set.
- **Tokens / Pay-per-use:** alternativa sin suscripción para usuarios casuales.

### Por qué tiempo real va en tier alto
- Más caro de servir (más cómputo, latencia, batería).
- Más difícil de copiar — barrera técnica real.
- Más valioso para atletas serios que entrenan solos.

### Sobre el tiempo real técnicamente

- Ya sembré la duda de "¿el usuario debe mirar la pantalla mientras hace sentadilla?" — y aplica para el usuario sin equipo. Pero hay un caso donde tu modelo premium SÍ tiene sentido:
Spotter virtual / Form check live — usuario tiene celular en trípode lateral, hace su set, el sistema le dice por audio (no visual): "rep 3, profundidad ok", "rep 4, perdiste profundidad", "rep 5, valgo en rodilla derecha". El usuario no mira la pantalla, escucha como si tuviera coach al lado.
Eso es premium real. Esa es la feature que valdría $30/mes.
